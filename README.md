# Dotfiles Managment #

Dotfiles currently managed as a repo by vcsh and myrepos.


Basic commands:

brew install vcsh mr

or 

apt-get install vcsh mr


vcsh vim add .vim
vcsh vim commit -m "inital commit"
vcsh vim remote add origin _githuburl_
vcsh vim push -u origin master
